# 业务卡：AI Token 销售平台（颠覆期收费站）（v0.4）

> 小甜心 2026-06-27 校准：从"P1 后置"升级为 **P1（与 ⑤ 并列）**
> 核心修正：小甜心逻辑成立——前期积累必经之路；行业没有头部 = 早期红利窗口
> 主归类：AI Center L5 项目层

## 1. 业务定义
- **一句话**：作为火山引擎正规代理商，向漫剧/短剧/视频工作室销售 AI Token（核心是 Seedance 2.0/2.5 等视频大模型的稀缺资源）
- **目标用户**：漫剧/短剧/视频工作室（to B）
- **核心价值**：年营收 100-500 万（早期）；500-2000 万（成熟期）

## 2. 行业判断（小甜心核心洞察）
- **判断 1**：视频生产处于颠覆期（Seedance 2.0 单月营收超 10 亿，2.5 已发布）
- **判断 2**：未来会产生大量视频生产工作室（漫剧/短剧为主）
- **判断 3**：这些工作室会大量消耗 Token（视频生成是重算力场景）
- **判断 4**：谁占据稀缺模型的 Token 通道，谁就能做"颠覆期收费站"
- **关键判断**：**行业没有头部，类比 3G 时代号段，代理商壁垒会随行业成熟自然形成**

## 3. 颠覆期收费站逻辑（核心）
**正面逻辑**：
- 早期 Token 价格战未白热化 → 赚稀缺性溢价
- 工作室获取 Token 困难 → 你提供"专业代理"
- 行业成熟后 → 代理壁垒自然形成（先入者优势）

**风险与对冲**：
- 90% 工作室死亡 → 但**这正是代理商的机会**（帮他们省钱 = 你的价值）
- 模型迭代快 → 但**你的护城河 = L3-L5 增值服务，不是 L1-L2 代理**
- 多模型竞争 → 但**你可以接多模型（火山 + 智谱 + 可灵）**

## 4. 5 层增值模式（核心护城河）
| 层 | 模式 | 利润 | 你的护城河 |
|---|---|---|---|
| L1 | Token 价差销售 | 薄 | 火山关系 |
| L2 | 套餐 + 模型组合 | 中 | 套餐设计 |
| L3 | **模型选型咨询** | **高** | **业务 ① 行业认知** |
| L4 | **自动化工作流搭建** | **高** | **业务 ⑤ 开发能力** |
| L5 | **成本归集 + 用量优化** | **极高** | **业务 ①+⑤ 组合** |

**核心策略**：从 L1 起步，**逐步向 L3-L5 升级**。**L3-L5 才是你的护城河**。

## 5. 启动前 5 项必答问题（硬条件）
1. **火山引擎代理条款**：保证金 / 销售目标 / 分成比例
2. **L3 增值服务设计**：模型选型咨询具体怎么收费
3. **L4 工作流搭建**：你或团队有工程能力
4. **L5 数据能力**：成本归集怎么做
5. **工作室客户验证**：已付费客户（不是潜在客户）

**5 项有 2 项不确定 → 业务 ⑥ 不启动**。

## 6. 涉及的项目
- 火山引擎代理谈判
- 公司注册（建议独立公司）
- L1 Token 销售基础平台
- L3 增值服务产品化
- L4 工作流模板开发
- 客户开发与成功服务

## 7. 涉及的业务间关系
- **依赖**：
  - **业务 ① 行业认知**（漫剧/短剧行业 = L3 护城河）
  - **业务 ⑤ 开发能力**（L4 工作流搭建）
  - IT 治理（基础设施）
- **被依赖**：
  - **业务 ② 私域**（Token 成本优势）
  - **业务 ③ KA 代运营**（L3 增值服务卖给 KA）
- **协同**：与 ①⑤ 共生（行业认知 + 工程能力是核心资产）

## 8. 业务生命周期
- 当前阶段：**筹划期**
- 启动时间：5 项启动条件确认后
- 预期成熟时间：12-24 个月

## 9. 退出标准
- 火山代理合作终止 + 无其他模型源 → 退出
- 工作室客户萎缩 → 转型或退出
- L3-L5 增值服务渗透率 < 10% → 转型为纯代理（护城河失效）
- 行业判断证伪（视频生产颠覆期未发生）→ 业务终止

## 10. 与火山的关系（重要原则）
- **不绑死火山**：你的行业判断独立于火山成立
- 火山是 L1-L2 代理入口，但 L3-L5 是你自己的护城河
- **核心原则**：火山是供应商之一，不是唯一供应商
- **关键动作**：保持与智谱 / 快手可灵 / 其他模型厂商的接洽
