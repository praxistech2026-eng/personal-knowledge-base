# 搜索工具矩阵

> AI Center 的搜索工具集合，覆盖精准单页抓取、大规模批量爬取、复杂反爬绕过、高匿名隐身浏览。

## 工具概览

| 工具 | 类型 | 用途 |
|------|------|------|
| Jina Reader | Skill | 单页精准抓取，干净 Markdown |
| Crawl4AI | Skill | 批量深度爬取，异步并行，JS 渲染 |
| Scrapling | pip | 复杂反爬，多引擎自适应解析 |
| CamoFox | pip | 高匿名隐身浏览器，指纹隐藏 |
| DuckDuckGo | CLI | 免费零成本兜底 |
| Tavily | pip | AI 语义优化搜索 |

## Jina Reader

| 属性 | 值 |
|------|-----|
| API Endpoint | `https://r.jina.ai/<url>` |
| 脚本 | `~/scripts/jina-read` |
| 认证 | Bearer Token（`jinja_` 开头） |
| 定位 | 单页快速抓取，内容质量高 |

## Crawl4AI

| 属性 | 值 |
|------|-----|
| Python 包 | `crawl4ai` v0.8.6 |
| 前置要求 | `playwright install chromium` |
| 定位 | 批量 URL、深层爬取、JS 渲染、LLM 过滤 |

## Scrapling

| 属性 | 值 |
|------|-----|
| Python 包 | `scrapling` v0.2.99 |
| 引擎 | Fetcher(httpx) / StealthyFetcher(Camoufox) / PlayWrightFetcher |
| 定位 | 复杂反爬场景，自适应解析 |

## CamoFox

| 属性 | 值 |
|------|-----|
| Python 包 | `camoufox` v0.4.11 |
| CLI | `~/.local/bin/camoufox` |
| 定位 | 最严格反爬，完整浏览器指纹隐藏 |

## DuckDuckGo

| 属性 | 值 |
|------|-----|
| 工具 | `ddgs` CLI |
| 定位 | 免费零成本兜底，隐私敏感时用 |

## Tavily

| 属性 | 值 |
|------|-----|
| Python 包 | `tavily` v0.7.24 |
| 限制 | 1000次/月免费额度 |
| 定位 | AI 语义优化搜索 |

## USAGE_MANUAL

### 健康检查

| 字段 | 值 | 说明 |
|------|----|------|
| 检查对象 | 工具包安装、脚本入口、外部搜索服务可用性 | 多工具组合 |
| 检查命令 | `python3 -c "import crawl4ai, scrapling, camoufox"`；各工具跑一个最小样例 | 导入 + 样例 |
| 判据 | 包可导入、脚本可跑、至少一条搜索链路可用 | 可用 |
| 频率 | 改依赖 / 改脚本 / 搜索失败时 | 变更即查 |
| 失败动作 | 先切到最稳的兜底工具（DDGS / Jina） | 先保可用 |

### 备份

| 字段 | 值 | 说明 |
|------|----|------|
| 备份对象 | 脚本、别名、依赖版本、代理配置 | 组合工具重建依赖 |
| 备份方式 | Git + 文档化环境重建 | 依赖可装 |
| 频率 | 变更后 | 版本留痕 |
| 保留策略 | 保留最近可重建说明 | 足够 |
| 恢复命令 | 重新安装依赖并按脚本复测 | 可重建即恢复 |

### 告警

| 字段 | 值 | 说明 |
|------|----|------|
| 告警条件 | 样例失败、反爬失败、搜索结果异常 | 任一即异常 |
| 通知渠道 | 当前无自动告警 | 先人工发现 |
| 兜底动作 | 切备用工具链 | 先保输出 |
| 升级路径 | 先恢复最稳工具，再调复杂工具 | 不硬扛 |

### 恢复 / 回滚

- 切回 DDGS / Jina Reader
- 重新安装依赖后复测
