# Whisper

> OpenAI 开源的 99+ 语言语音识别模型，本地部署。

## 官方资源

| 资源 | 链接 |
|------|------|
| GitHub (OpenAI) | https://github.com/openai/whisper |
| Python 包 | `openai-whisper` (pip) |

## 功能

- **99+ 语言识别**：支持普通话、英语、日语、韩语等
- **多模型规格**：tiny / base / small / medium / large
- **本地推理**：无需联网，保护隐私
- **字幕生成**：可输出 SRT/VTT 格式

## 在本生态中的角色

**语音识别层**。Whisper 是 AI Center 的本地 ASR 引擎，用于将语音输入转为文字，供 Agent 处理。

## USAGE_MANUAL

### 健康检查

| 字段 | 值 | 说明 |
|------|----|------|
| 检查对象 | Python 包、模型文件、转写脚本 | 无常驻进程 |
| 检查命令 | `python3 -c "import whisper"`；运行一次短音频转写 | 导入 + 实测 |
| 判据 | 能导入且能产出转写结果 | 可用 |
| 频率 | 改模型 / 改 venv / 改脚本后 | 变更即查 |
| 失败动作 | 先补依赖，再确认模型文件是否存在 | 先环境后模型 |

### 备份

| 字段 | 值 | 说明 |
|------|----|------|
| 备份对象 | venv 说明、模型下载路径、调用脚本 | 真正需要留的是环境说明 |
| 备份方式 | Git + 文档化重建步骤 | 依赖可重装 |
| 频率 | 变更后 | 版本留痕 |
| 保留策略 | 保留可重建说明即可 | 不备份大模型文件到文档 |
| 恢复命令 | 重新安装 `openai-whisper` 并复测 | 可重建即恢复 |

### 告警

| 字段 | 值 | 说明 |
|------|----|------|
| 告警条件 | 导入失败、模型不存在、转写报错 | 任一即异常 |
| 通知渠道 | 暂未接入独立自动告警 | 现由巡检/人工发现补位 |
| 兜底动作 | 切回云端/备用转写工具 | 保证先可用 |
| 升级路径 | 先修本地环境，再恢复默认工具链 | 不硬扛 |

### 恢复 / 回滚

- 重新安装 `openai-whisper`
- 重新下载模型并复测转写
