### 健康检查

| 字段 | 值 | 说明 |
|------|----|------|
| 检查对象 | `5678` 端口 / n8n 进程 | 当前档案状态是“已安装未运行” |
| 检查命令 | `ss -tlnp | grep 5678`；`ps aux | grep n8n` | 端口与进程 |

### 备份

| 字段 | 值 | 说明 |
|------|----|------|
| 备份对象 | n8n 工作流、凭证与本机配置（具体落点后续补核） | 当前未固化独立目录 |

### 告警

| 字段 | 值 | 说明 |
|------|----|------|
| 告警条件 | 启用状态异常、5678 不可达、导入失败 | 任一即异常 |
| 通知渠道 | 当前无自动告警 | 现状如实写 |
| 兜底动作 | 先区分“未启用”还是“故障” | 不把状态混淆 |
| 升级路径 | 先本机确认，再补告警链路 | 后续自动化 |

### 恢复 / 回滚
