### Step 2：找 Homepage 容器和挂载的 yaml 路径

```bash
docker ps | grep -i homepage
# 假设容器名是 homepage，主机挂载路径假设是 /opt/homepage 或 /home/<user>/homepage
# 容器内配置目录固定是 /app/config
# 找挂载源：
docker inspect homepage --format '{{json .Mounts}}' | python3 -m json.tool
# 找带 /app/config 的那行，看 Source 字段
```

假设挂载源是 `/opt/homepage/config/`（实际以 docker inspect 输出为准），后续命令按这个假设写。

### Step 3：修改 services.yaml（AList → OpenList）

```bash
cd <挂载源路径>
cp services.yaml services.yaml.bak.$(date +%Y%m%d)
sed -i 's/- AList:/- OpenList:/' services.yaml
sed -i 's/abbr: AL/abbr: OL/' services.yaml
sed -i 's|网盘聚合器|网盘聚合器（v4.2.2 OpenList）|' services.yaml
```

### Step 4：修改 bookmarks.yaml（同样）

```bash
cp bookmarks.yaml bookmarks.yaml.bak.$(date +%Y%m%d)
sed -i 's/- AList:/- OpenList:/' bookmarks.yaml
sed -i 's/abbr: AL/abbr: OL/' bookmarks.yaml
```

### Step 5：重启 Homepage 容器让它重读 yaml

```bash
docker restart homepage
# 等 5 秒
sleep 5
curl -s http://127.0.0.1:3000/api/services | python3 -c "
import json, sys
d = json.load(sys.stdin)
for g in d:
    for s in g.get('services', []):
        if 'OpenList' in s.get('name', '') or 'openlist' in s.get('href', '').lower():
            print(f\"✅ OpenList 已生效: {s}\")
            break
"
```

## 长期建议

### 方案 A：在 254 上设免密 SSH（推荐）

```bash
# 在 AI Center (249) 执行
ssh-keygen -t ed25519 -f ~/.ssh/homepage_sync -N ""
# 把公钥复制到 254
ssh-copy-id -i ~/.ssh/homepage_sync.pub root@192.168.2.254
# 之后可以免密同步
scp -i ~/.ssh/homepage_sync services.yaml root@192.168.2.254:/opt/homepage/config/
```

之后任何 AI Center 这边的修改都能一条命令同步过去。

把 254 上的 Homepage 容器迁移到 192.168.2.249：
