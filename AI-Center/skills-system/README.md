## USAGE_MANUAL

### 健康检查

| 字段 | 值 | 说明 |
|------|----|------|
| 检查对象 | `~/.hermes/skills/`、`skills list`、角色库目录 | 技能库存本体 |
| 检查命令 | `skills list`；检查目录是否完整 | 库与索引 |
| 判据 | 技能可列出、目录存在、关键角色库完整 | 可用 |
| 频率 | 安装/更新技能后 | 变更即查 |
| 失败动作 | 先恢复缺失目录，再检查索引 | 先目录后索引 |

### 备份

| 字段 | 值 | 说明 |
|------|----|------|
| 备份对象 | `~/.hermes/skills/`、`agency-agents` 目录、参考文档 | 核心资产 |
| 备份方式 | Git 同步 + 目录快照 | 技能可版本化 |
| 频率 | 改技能 / 加角色后 | 版本留痕 |
| 保留策略 | 保留最近可回滚版本 | 足够恢复 |
| 恢复命令 | 重新同步 skills 目录并复测 `skills list` | 回滚后验 |

### 告警

| 字段 | 值 | 说明 |
|------|----|------|
| 告警条件 | 技能缺失、角色库损坏、列表数量异常 | 任一即异常 |
| 通知渠道 | 当前无自动告警 | 先人工发现 |
| 兜底动作 | 先恢复目录，再修引用 | 先保可用 |
| 升级路径 | 先恢复技能库存，再做归档 | 不硬扛 |

### 恢复 / 回滚

- 恢复 `~/.hermes/skills/`
- 重新 `skills list` 验证数量与可见性
