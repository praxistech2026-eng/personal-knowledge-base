### [1777699056] assistant

没有 App 的话，首次登录需要通过浏览器认证。有两种方式：

**方式 1：浏览器 OAuth（推荐）**
```bash
brew install tailscale
tailscale login
```
这会自动打开浏览器，用 Tailscale 账号（你服务器上用的同一个账号）授权即可。

**方式 2：从已登录设备获取 auth key**

1. 在 Tailscale 管理后台（[console.tailscale.com](https://console.tailscale.com)）→ **Settings** → **Keys** → **Generate auth key**
2. 然后在 Mac 上：
```bash
tailscale login --authkey <生成的key>
```

---

你服务器上已经在用 Tailscale 了，方式 1 最简单。浏览器授权完成后，Mac 节点会自动出现在你的 Tailscale 网络里。
