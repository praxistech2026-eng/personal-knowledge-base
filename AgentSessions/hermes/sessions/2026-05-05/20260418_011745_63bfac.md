# Attention 知识卡片

## Summary

**Attention** 是深度学习中的一种机制，最早在 2014 年用于图像生成描述，2017 年在《Attention Is All You Need》论文中被发扬光大，成为 Transformer 架构的核心。它让模型能够**动态地关注输入序列中与当前任务最相关的部分**，克服了传统 RNN 的长距离依赖问题，大幅提升了并行计算效率。

---

## Key Points

- **核心思想**：对序列中每个位置计算"注意力权重"，加权求和得到上下文向量
- **Scaled Dot-Product Attention**：`Attention(Q,K,V) = softmax(QK^T / √d_k) V`
- **Multi-Head Attention**：多个注意力头并行工作，捕捉不同子空间的关系
- **Self-Attention**：Q、K、V 均来自同一输入，捕捉序列内部依赖
- **Cross-Attention**：Q 来自目标序列，K/V 来自源序列，用于 seq2seq 任务
- **优点**：并行化友好、可捕捉长距离依赖、可解释性强
- **缺点**：计算复杂度 O(n²)，对长序列内存开销大
