# Attention机制

## 🧠 Summary
让模型在处理序列数据时能够动态关注最重要部分的技术，通过计算Query与Key的相似度分配权重，对Value加权求和。

## 🧩 Primary Concept
**Query-Key-Value (QKV) 加权求和** — 输入通过三个线性变换生成Query、Key、Value，Query与所有Key计算注意力分数，经Softmax归一化后对Value加权求和。

## 🔑 Key Points
- **核心公式**: Attention(Q,K,V) = Softmax(QK^T / √d_k) × V
- **解决长距离依赖**: 任意位置直接关联，无信息逐层传递损失
- **并行计算**: 矩阵运算支持GPU高效加速，突破RNN顺序限制
- **多头机制**: 多个注意力头并行，捕捉不同子空间特征（Transformer）
- **自注意力**: Query、Key、Value同源（自身序列），实现内部交互
