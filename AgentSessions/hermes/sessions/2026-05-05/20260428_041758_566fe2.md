## 原因

Claude Code 默认对**所有操作**都要求确认，包括：
- 网络查询（即使只是搜索）
- 读取文件
- 执行命令

## 解决方法

在你项目根目录创建 `.claude/settings.json`（不是全局的，是项目级的），加入：

```json
{
  "permissions": {
    "allow": ["WebSearch", "Read"],
    "ask": ["Edit", "Write"],
    "deny": ["Bash"]
  }
}
```

或者更简单——直接修改项目的 `.claude/permissions` 文件：

```
allow web-search
allow read
ask edit
ask write
deny bash
```

```json
{
  "permissions": {

}
```

- **allow** — 自动执行，不询问
- **ask** — 每次询问（默认）
