# 位置编码 Position Encoding

## 🧠 Summary
位置编码是一种将序列中token位置信息注入Transformer模型的技术，使模型能够区分不同位置的词元。

## 🧩 Primary Concept
在自注意力机制中引入token的顺序信息，解决Transformer本身无法感知序列顺序的问题。

## 🔑 Key Points
- **正弦/余弦位置编码**（Vaswani et al., 2017）：通过不同频率的正弦和余弦函数生成，公式为 `PE(pos,2i) = sin(pos/10000^(2i/d))`
- **绝对位置编码**：为每个位置分配固定的绝对嵌入，与token内容嵌入相加输入模型
- **相对位置编码**：编码token之间的相对距离而非绝对位置，效果更好但实现复杂
- **旋转位置编码 RoPE**（Su et al., 2022）：通过旋转矩阵将位置信息编码进Query和Key，是LLaMA等模型采用
- **ALiBi**（Press et al., 2021）：基于token距离的线性偏置，无需学习的位置编码
