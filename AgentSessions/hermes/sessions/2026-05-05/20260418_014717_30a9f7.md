# Transformer模型

## 🧠 Summary
一种基于自注意力机制的深度学习架构，革新了自然语言处理并成为现代大语言模型的基础。

## 🧩 Primary Concept
**自注意力机制（Self-Attention）**：通过计算序列内任意位置元素之间的关联性，实现并行处理和长距离依赖建模。

## 🔑 Key Points
1. **起源**：2017年Google论文《Attention Is All You Need》，完全抛弃RNN/CNN结构
2. **核心组成**：多头自注意力层 + 前馈神经网络 + 残差连接 + 层归一化
3. **关键创新**：位置编码（Positional Encoding）保留序列顺序信息
4. **并行优势**：相比RNN可高效并行训练，彻底解决长距离依赖问题
5. **衍生模型**：BERT（双向Encoder）、GPT系列（Decoder-only）、T5（Encoder-Decoder）
