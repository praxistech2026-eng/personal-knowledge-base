# Attention机制

## 🧠 Summary
Attention机制是一种让模型在处理信息时能够"关注"最相关部分的技术，通过动态分配权重来权衡不同位置信息的重要性。

## 🧩 Primary Concept
**注意力权重**——模型自动学习并应用一组权重，决定在生成每个输出时应该给予输入各部分多大的关注。

## 🔑 Key Points
- **核心思想**：不再将整个输入压缩为固定长度向量，而是允许输出阶段访问所有输入位置
- **工作流程**：计算Query与所有Key的相似度 → Softmax归一化得到注意力权重 → 用权重对Value加权求和
- **优势**：解决长序列依赖问题，提升模型对关键信息的捕捉能力
- **类型**：Self-Attention（自注意力）、Multi-Head Attention（多头注意力）、Cross-Attention（跨注意力）
- **应用**：Transformer架构的核心组件，广泛用于NLP、CV、多模态等领域
