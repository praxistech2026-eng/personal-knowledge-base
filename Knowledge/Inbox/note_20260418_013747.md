     # Transformer 模型                                                         
                                                                                
     ## 🧠 Summary                                                              
     一种基于自注意力机制处理序列数据的深度学习架构，摒弃了传统的循环神经网     
     络结构，能够并行处理序列中的任意位置关系。                                 
                                                                                
     ## 🧩 Primary Concept                                                      
     **自注意力机制（Self-Attention）**：通过计算序列中每个位置对其他所有位     
     置的注意力权重，捕捉长距离依赖关系。                                       
                                                                                
     ## 🔑 Key Points                                                           
     - **核心组件**：编码器（Encoder）+ 解码器（Decoder），或仅编码器（如       
     BERT）或仅解码器（如 GPT）                                                 
     - **多头注意力（Multi-Head                                                 
     Attention）**：并行运行多个注意力机制，捕捉不同子空间特征                  
     - **位置编码（Positional                                                   
     Encoding）**：由于无循环结构，需通过位置编码注入序列顺序信息               
     - **前馈神经网络（FFN）**：每个注意力层后接全连接层进行非线性变换          
     - **优势**：支持并行计算、捕获长距离依赖、训练稳定性强于 RNN
