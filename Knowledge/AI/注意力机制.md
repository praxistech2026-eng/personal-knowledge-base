# 注意力机制

## 🧠 Summary
一种让模型在处理信息时动态关注最重要部分的技术。

## 🧩 Canonical Concept
注意力机制

## 🧩 Primary Concept
通过计算-query 与 key 的相关性，为 value 分配不同权重，使模型聚焦关键信息。

## 🔑 Key Points
- 核心三要素：Query（查询）、Key（键）、Value（值）
- 通过 Softmax 归一化生成注意力权重
- 权重与 Value 加权求和得到输出
- 突破序列依赖，允许任意位置信息直接交互
- 是 Transformer 架构的核心组件
