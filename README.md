# HermesKnowledge OS

个人知识管理系统。

## 快速开始

- **新建知识**：`hn "你的知识内容"`
- **查看手册**：[USAGE_MANUAL.md](./USAGE_MANUAL.md)
- **自动化系统**：[.system/README.md](./.system/README.md)

## 目录

| 目录 | 说明 |
|------|------|
| `Knowledge/` | 知识卡片 |
| `AgentSessions/` | Agent 会话归档 |
| `Inbox/` | 临时收集 |
| `.system/` | 自动化系统（SemVer 版本控制） |

## 同步

笔记数据实时同步到 GitHub（`praxistech2026-eng/hermes-knowledge-base`）。
Mac 端 Obsidian Git 插件自动拉取。
